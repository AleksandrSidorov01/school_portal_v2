# Создание профиля учителя

## Способ 1: Через скрипт (РЕКОМЕНДУЕТСЯ)

Используйте скрипт для быстрого создания учителя:

```bash
npm run create-teacher
```

### С параметрами:

```bash
npm run create-teacher <email> <пароль> <имя> <фамилия> <специализация> [номер_сотрудника] [телефон]
```

### Примеры:

**Минимальный (с значениями по умолчанию):**
```bash
npm run create-teacher
```
Создаст учителя:
- Email: `teacher@school.com`
- Пароль: `teacher123`
- Имя: `Учитель Тестовый`
- Специализация: `Математика`

**С указанием всех параметров:**
```bash
npm run create-teacher ivanov@school.com ivanov123 Иван Иванов Математика EMP001 +79991234567
```

**Только email и пароль:**
```bash
npm run create-teacher petrov@school.com petrov123
```

## Способ 2: Через API (требует роль ADMIN)

Если у вас есть доступ к админ панели или вы можете делать API запросы:

1. **Сначала создайте пользователя с ролью TEACHER** (через админ панель или напрямую в БД)

2. **Затем создайте профиль учителя:**
```bash
POST http://localhost:3000/api/teachers
Authorization: Bearer <admin_token>
Content-Type: application/json

{
  "userId": "<user_id>",
  "specialization": "Математика",
  "employeeNumber": "EMP001",
  "phone": "+79991234567"
}
```

## Способ 3: Через Prisma Studio

1. Откройте Prisma Studio:
```bash
npm run prisma:studio
```

2. Создайте пользователя в таблице `users` с ролью `TEACHER`

3. Создайте профиль учителя в таблице `teachers`, указав `userId`

## Важно!

- Пользователь должен иметь роль `TEACHER` перед созданием профиля
- Email должен быть уникальным
- После создания профиля учитель сможет войти в систему и использовать функционал для учителей

## Проверка

После создания учителя:
1. Войдите в систему с email и паролем учителя
2. Вы должны увидеть панель учителя с возможностью:
   - Выставлять оценки
   - Просматривать классы
   - Просматривать расписание
