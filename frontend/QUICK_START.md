# Быстрый старт

## Шаг 1: Запуск Backend

В корне проекта:
```bash
npm run dev
```

Backend должен запуститься на `http://localhost:3000`

## Шаг 2: Запуск Frontend

В папке `frontend`:
```bash
cd frontend
npm run dev
```

Frontend должен запуститься на `http://localhost:5173`

## Шаг 3: Открытие в браузере

1. Откройте браузер (Chrome, Firefox, Edge)
2. В **адресной строке** (не в поиске!) введите:
   ```
   http://localhost:5173
   ```
3. Нажмите Enter

## Что должно произойти

- Если вы не авторизованы → откроется страница **Входа**
- Если вы авторизованы → откроется **Дашборд**

## Создание первого пользователя

1. Откройте `http://localhost:5173/register`
2. Заполните форму регистрации
3. Выберите роль (STUDENT, TEACHER или ADMIN)
4. После регистрации вы будете автоматически перенаправлены на дашборд

## Проверка работы Backend

Откройте в браузере:
```
http://localhost:3000/api/health
```

Должен вернуться JSON:
```json
{
  "status": "ok",
  "message": "Школьный портал работает!"
}
```

## Решение проблем

### Backend не запускается (порт 3000 занят)

```bash
npm run stop
```

Или найдите процесс:
```bash
netstat -ano | findstr :3000
taskkill /PID <номер_процесса> /F
```

### Frontend не подключается к Backend

1. Проверьте, что backend запущен на порту 3000
2. Создайте файл `.env` в папке `frontend`:
   ```env
   VITE_API_URL=http://localhost:3000/api
   ```
3. Перезапустите frontend сервер

### Страница не открывается

- Убедитесь, что вводите адрес в **адресной строке браузера**, а не в поиске
- Проверьте, что оба сервера (backend и frontend) запущены
- Откройте консоль браузера (F12) и проверьте ошибки
