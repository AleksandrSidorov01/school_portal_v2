# Исправление проблемы с подключением к Frontend

## Проблема
Браузер не может подключиться к `localhost:5173`, хотя Vite показывает, что сервер запущен.

## Решение

### Шаг 1: Остановите текущий Vite сервер
В терминале, где запущен `npm run dev`, нажмите:
```
Ctrl + C
```

### Шаг 2: Перезапустите сервер
```bash
npm run dev
```

### Шаг 3: Попробуйте открыть с полным URL
В браузере откройте:
```
http://127.0.0.1:5173
```
или
```
http://localhost:5173
```

**Важно:** Используйте `http://` в начале!

## Альтернативное решение

Если проблема сохраняется, попробуйте:

1. **Использовать другой порт:**
   ```bash
   npm run dev -- --port 3001
   ```

2. **Проверить firewall:**
   - Убедитесь, что Windows Firewall не блокирует порт 5173

3. **Очистить кеш браузера:**
   - Нажмите `Ctrl + Shift + Delete`
   - Очистите кеш и cookies

4. **Попробовать другой браузер:**
   - Firefox, Edge, или Chrome в режиме инкогнито

## Проверка

После перезапуска в консоли должно быть:
```
VITE v7.2.2 ready in XXX ms
Local: http://localhost:5173/
Network: http://192.168.x.x:5173/
```

Если видите "Network" адрес - это хорошо, сервер доступен!
